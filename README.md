
# Binance Futures Trading Bot

A Python-based CLI trading bot for Binance Futures Testnet (USDT-M). Supports Market, Limit, Stop-Limit, and TWAP algorithmic orders.

## Features
- **Core:** Market and Limit orders with validation.
- **Advanced:** Stop-Limit protection and TWAP (Time-Weighted Average Price) execution.
- **System:** Structured logging to `bot.log` and error handling.
